# ELECTRONIC-VEHICLE-CHARGERS

It an server appliction to store details of Charging stations in different Locations and the connectors it contain... it has basic api routes to perform get/post/delete operations and mongodb is used as the database

# Setup

## create and add the following in the .env file
- PORT=5000
- MONGODB_URI=

## To run(dev server)
- npx nodemon server.js
